Number Of Overridden Methods (NOOM)

This metric counts the number of methods overridden in the class from superclasses or interfaces.

Included in the set of metrics proposed by M. Lorenz and J. Kidd, see

M. Lorenz, J. Kidd, “Object Oriented Software Metrics”, Prentice Hall, NJ, 1994.

According to

Filó, T. G., Bigonha, M., & Ferreira, K. 2015. A catalogue of thresholds for object-oriented software metrics. Proc. of the 1st SOFTENG, 48–55.

common value for this metric is less than 3, casual - between 3 and 4, uncommon is greater than 4.
